# Mustache Core Library for Android

###### Add this to your root build.gradle
```bash
allprojects {
  repositories {
    maven { url 'https://jitpack.io' }
  }
}
```

###### Add this to your root build.gradle
```bash
dependencies {
  implementation 'com.github.mustachedk:core-lib-an:0.1.4'
}
```

#### Currently Contains:
##### Utilities
* LocationUtil
* NetworkUtil
* PermissionUtil
* PixelUtil
* ScreenUtil
* ValidateUtil

##### Base Classes:
* DataBindingAdapter (DataBindingViewHolder)
